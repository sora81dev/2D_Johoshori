# 2D_Johoshori
岐阜高専2026年度　電子制御工学科　通年科目　必修履修である「情報処理Ⅰ」のデータです。  
以下の注意を守る場合のみ利用可能です。

## 実行環境

> [!IMPORTANT]
> clangではなくgccを使用している点に注意してください。

|項目|値|
|-|-|
|CPU Arch.|ARM64|
|IDE|neovim|
|gcc|gcc (stable 15.2.0)|

## 使用する際の注意
- レポートのデータが含まれています。コピーなどは避け、参考にする程度にしてください
- プログラムは確実に動作するとは限りません。自己責任で実行してください。
